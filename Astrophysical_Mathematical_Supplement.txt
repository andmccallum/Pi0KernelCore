SECTION A – PRECISE GRAVITATIONAL FIELD EQUATIONS
================================================================================

A.1 Einstein Field Equations (Component Form)

    R_μν - (1/2)g_μν R + Λg_μν = (8πG/c⁴) T_μν

Expanded components in spherical coordinates (t,r,θ,φ):

    R_tt = -(1/2r²) d/dr[r(1-2GM/rc²)] + Λ
    R_rr = (GM/r²c²)(1-2GM/rc²)⁻¹ + Λ  
    R_θθ = r²[1 + GM/rc² - Λr²/3]
    R_φφ = R_θθ sin²θ

A.2 Schwarzschild Metric

    ds² = -(1-2GM/rc²)c²dt² + (1-2GM/rc²)⁻¹dr² + r²dΩ²

A.3 Kerr Metric (Rotating Black Hole)

    ds² = -(1-2GMr/ρ²c²)c²dt² + (4GMar sinθ/ρ²c)dtdφ + 
          (ρ²/Δ)dr² + ρ²dθ² + (r²+a²cos²θ)sin²θ dφ²

where:
    ρ² = r² + a²cos²θ
    Δ = r² - 2GMr/c² + a²
    a = J/(Mc) = angular momentum parameter
